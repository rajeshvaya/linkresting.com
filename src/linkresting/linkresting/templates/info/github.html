Github

Entire code of linkresting is hosted at Github, You can setup your own linkresting or fork it to contribute


Contribution

Linkresting is developed entirely in Python and Vagrant. To contribute to linkresting:

  • Install Vagrant (if you don't already have it)
  • Fork the repositary and clone to your local machine
  • Run "vagrant up"
  • Switch to virtualenv "default" by using "workon default"
  • Contribute to the /src
  • Merging the code is only a pull request away
